Course Description

This course introduces the fundamental concepts of data structures and the algorithms that operate on them. Topics include data abstraction, recursion, stacks, queues, linked lists, trees, hash tables, graphs, and analysis of algorithms. Students will implement various data structures and algorithms in programming assignments.
