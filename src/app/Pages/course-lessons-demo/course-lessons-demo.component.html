Course Lessons Demo

This demo shows how to use the course lessons component with the API endpoint.

Demo Controls

API Information

Endpoint:

GET /lesson/course-lessons

Parameters:

  • courseId: {{ selectedCourseId }}
  • pageNumber: Dynamic (based on pagination)
  • pageSize: {{ pageSize }}
  • orderBeforPagination: true/false
  • orderDirection: 0 (Ascending) / 1 (Descending)
